# 🏗️ System Design Notes

Welcome to the **System Design Notes** repository! 🚀

This repository contains structured, easy-to-understand notes that I have compiled while learning System Design (both HLD and LLD) from various amazing **YouTube playlists** and tutorials. It is designed to be a one-stop quick reference for core architectural concepts, software engineering interview prep, and clean coding principles.

## 📂 Repository Structure

### 1. [High-Level Design (HLD)](./HLD_Notes.md)
This section focuses on macro-architecture. It contains notes on how large-scale distributed systems are designed, how different components interact, scalability tradeoffs, databases, and general system design concepts covered across popular YouTube series.

### 2. [Low-Level Design (LLD)](./LLD_Notes.md)
This section dives into micro-architecture. Here you will find compiled notes on Object-Oriented Design (OOD), SOLID principles, design patterns, and clean code practices along with code snippets.

## 💡 How to Use
Feel free to explore the markdown files directly. Since these notes are synthesized from video lectures, they summarize complex architectural topics into digestible text formats. They are great for quick revisions before technical interviews!
